# Poción de Caída Suave Mod

## Descripción
El **Poción de Caída Suave Mod** es un mod para **Minecraft 1.19.2** con **Forge**, que introduce una nueva poción al juego: la *Poción de Caída Suave*.

## Características
- Al beber la poción, el jugador obtiene el efecto de **Absorción de Caída** durante **30 segundos**.
- Este efecto **previene el daño por caída**, permitiendo explorar alturas sin riesgos.
