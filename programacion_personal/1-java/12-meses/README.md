## Plan de estudio 12 meses — Camino firme para programador junior

### Mes 1-2: Fundamentos sólidos de Java

* Variables, tipos, operadores.
* Estructuras de control (`if`, `switch`, bucles).
* Arrays y colecciones básicas (`ArrayList`, `HashMap`).
* Métodos y recursividad sencilla.
* Proyecto: Gestor de tareas por consola.

### Mes 3-4: Programación Orientada a Objetos (POO)

* Clases, objetos, atributos, métodos.
* Modificadores de acceso, encapsulación.
* Constructores y sobrecarga.
* Herencia, polimorfismo.
* Proyecto: Sistema de biblioteca (libros, usuarios, préstamos).

### Mes 5-6: Bases de datos con SQL y JDBC

* Conceptos básicos de bases de datos.
* Consultas SQL: SELECT, INSERT, UPDATE, DELETE.
* Relaciones y JOINs.
* Integración con Java usando JDBC.
* Proyecto: Biblioteca conectada a base de datos.

### Mes 7-8: Java avanzado y buenas prácticas

* Manejo de excepciones.
* Archivos (lectura y escritura).
* Interfaces y clases abstractas.
* Colecciones avanzadas (`Set`, `Queue`).
* Proyecto: Gestor de inventario con persistencia.

### Mes 9: Fundamentos web básicos

* HTML5: estructura, formularios, tablas.
* CSS básico y responsive.
* JavaScript: DOM, eventos.
* Proyecto: Página web simple con formulario.

### Mes 10-11: Backend + frontend

* Crear API REST sencilla en Java (por ejemplo con Spring Boot).
* Conectar frontend web con backend usando fetch/AJAX.
* Proyecto: Mini CRM o gestor de gastos con frontend y backend.

### Mes 12: Proyecto final completo

* Integrar todo lo aprendido.
* Documentar código y subir a GitHub.
* Preparar presentación o demo para prácticas o empleo.

---

**Consejos para todo el año:**

* Dedica al menos 1 hora diaria o 5 días a la semana.
* Documenta todo en GitHub.
* Ve haciendo preguntas concretas y resolviendo dudas puntuales (aquí te puedo ayudar).
* No te compares con nadie, ve a tu ritmo.
