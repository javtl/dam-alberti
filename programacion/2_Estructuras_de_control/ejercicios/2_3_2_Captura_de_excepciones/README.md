# EJERCICIOS: Capturas de Excepciones

Ejercicio 2.3.1
Escribir un programa que pregunte al usuario su edad y muestre por pantalla todos los años que ha cumplido (desde 1 hasta su edad).

Ejercicio 2.3.2
Escribir un programa que pida al usuario un número entero positivo y muestre por pantalla todos los números impares desde 1 hasta ese número separado por comas.

Ejercicio 2.3.3
Escribir un programa que pida al usuario un número entero positivo y muestre por pantalla la cuenta atrás desde ese número hasta cero separados por comas deberá solicitar el número hasta introducir el correcto.

Ejercicio 2.3.4
EScribir un programa que pida al usuario un número entero, si la entrada no es correcta mostrará el mensaje "la entrada no es correcta" y lanzará 
la excepción capturada

Ejercicio 2.3.5
Escribir que solicite una contraseña y si no coincide con la que se tiene lance una excepción NameError con el mensaje "incorerct Password!!"
