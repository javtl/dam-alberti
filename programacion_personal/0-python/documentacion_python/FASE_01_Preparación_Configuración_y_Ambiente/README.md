## FASE 1: Preparación, Configuración y Ambiente

El punto inicial de nuestro estudio se centra en establecer el entorno de trabajo y la introducción más básica al lenguaje.

### Puntos Clave a Estudiar:

#### 1. Decidir el Punto de Partida: Guía para Principiantes

Dado que estás empezando este año en programación, el primer punto fundamental es seguir la recomendación de la propia documentación.

*   El sitio oficial de Python facilita empezar y usar Python.
*   La documentación incluye una **Guía para principiantes** (*Beginner’s Guide*) listada bajo "Otros recursos".
*   Si eres nuevo en la programación, la documentación te recomienda **"Start with our Beginner’s Guide"** (Comenzar con nuestra Guía para principiantes).
*   Es importante notar que el recurso principal, **"El tutorial de Python"**, está diseñado para *programadores* que son nuevos en el lenguaje Python, **no** *principiantes* que son nuevos en la programación, ya que espera que tengas una comprensión básica de la programación en general.

#### 2. Instalación, Configuración y Uso

Para poder practicar de manera efectiva, debes revisar la sección de la documentación dedicada al manejo del entorno.

*   Esta información se encuentra en la sección **"Python setup and usage"** (Configuración y uso de Python), que describe "How to install, configure, and use Python" (Cómo instalar, configurar y usar Python).

#### 3. Introducción General al Lenguaje

Una vez que tengas tu ambiente configurado, puedes empezar con el vistazo general al lenguaje.

*   El primer capítulo del tutorial es **"1. Abriendo el apetito"**. Este sirve como un resumen general.
*   Python es un lenguaje de programación potente y fácil de aprender.
*   Tiene estructuras de datos de alto nivel eficientes y un simple, pero efectivo, sistema de programación orientado a objetos.
*   Su elegante sintaxis y tipado dinámico, junto con su naturaleza interpretada, lo convierten en un lenguaje ideal para *scripting* y desarrollo rápido de aplicaciones en muchas áreas.

### Ejercicio Práctico Inicial (Paso 1 y 2 de la Checklist)

Para asegurarnos de que la configuración y el entorno están listos, el primer ejercicio se centra en el uso del intérprete, tal como se aborda en el **Capítulo 2. Usando el intérprete de Python**.

**Tarea de Práctica:**

1.  **Confirma tu Versión:** Instala Python (o confirma que lo tienes instalado) según la sección **Python setup and usage**.
2.  **Invoca el Intérprete:** Abre la terminal o línea de comandos e invoca el intérprete de Python (por ejemplo, escribiendo `python` o `python3`). La documentación menciona que **tener un intérprete a mano es útil para la experiencia práctica**.
