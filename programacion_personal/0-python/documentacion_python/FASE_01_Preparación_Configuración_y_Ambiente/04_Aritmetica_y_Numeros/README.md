## FASE 1: Paso 3 - Aritmética y Números

### Puntos Clave a Estudiar (Capítulo 3.1.1. Números)

Esta sección se encuentra bajo **3. Una introducción informal a Python** y específicamente **3.1. Usando Python como una calculadora: 3.1.1. Números**. Las matemáticas simples son sencillas con Python, y la sintaxis de la expresión es clara.

1.  **Operadores Aritméticos Estándar:** Los operadores `+`, `-`, `*` y `/` funcionan como se espera. Los paréntesis `()` se pueden usar para agrupar operaciones.

2.  **Tipos de División:**
    *   **División Clásica (`/`):** Esta división siempre devuelve un número de punto flotante (*float*). Por ejemplo, en la documentación se muestra que `17 / 3` devuelve `5.666666666666667`.
    *   **División de Piso (`//`):** Esta división (conocida como *floor division*) descarta la parte fraccionaria y devuelve solo la parte entera. Por ejemplo, `17 // 3` devuelve `5`.

3.  **Exponenciación:** Se utiliza el operador `**` para calcular potencias (exponenciación). Por ejemplo, `2 ** 3` devuelve `8`.

4.  **Uso en Modo Interactivo:** La práctica de estas operaciones se realiza idealmente en el **modo interactivo** del intérprete de Python, el cual se recomienda tener "a mano para la experiencia práctica".
